# authorityeditor

Authority Editor, on a modern stack.

## Build for web

flutter build web

In build/web folder:

git add .
git commit -m "fresh build"
git push origin main


If wiped:
git init
git add .
git commit -m "fresh build"
git remote add origin git@github.com:richardlehane/authorityweb.git
git push --force origin main
